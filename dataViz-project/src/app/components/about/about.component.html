Context

The Russo-Ukrainian conflict visualization site responds to the growing need for understanding and easy access to events related to this complex conflict. Faced with the overabundance of scattered, often contradictory information, and the difficulty for users to follow the temporal evolution of events, this project aims to provide an interactive and informative platform.
By combining an interactive map, a timeline, media related to each event, and an activity heatmap, the site aims to offer a complete user experience to understand and follow in a clear and objective manner the evolution of the Russo-Ukrainian conflict, while encouraging fact-checking from reliable sources.

Data Sources

This data provides detailed geographic information about the regions of both countries (Russia & Ukraine), which is essential for creating an accurate interactive map. They allow events to be visualized in their spatial context. Pie Chart

The pie chart allows you to visualize the proportions of events depending on their type (bombing, military drone, etc.) and the region over a defined period.
A filter is carried out on each type of event represented on the pie chart. This filter is applied to the map in all regions linked to the conflict.

Stacked bars

The stacked bar chart represents several series of events in bar form, with each bar subdivided into stacked segments for each country.
It provides information on the number of events linked to the conflict, having taken place in each country over a given period and also gives the number of ancillary events unrelated to the conflict.
This allows you to visualize the contribution of each category to the total, as well as the variations within each category over a given period.
